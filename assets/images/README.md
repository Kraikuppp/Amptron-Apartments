# Hero Background Image

## การใช้รูปพื้นหลังสำหรับ Hero Section

### วิธีการเพิ่มรูปพื้นหลัง

1. วางไฟล์รูปภาพของคุณในโฟลเดอร์ `assets/images/`
2. ตั้งชื่อไฟล์เป็น `hero-bg.jpg` หรือแก้ไขชื่อไฟล์ใน CSS

### รูปแบบไฟล์ที่รองรับ
- JPG/JPEG
- PNG
- WebP

### ขนาดที่แนะนำ
- ความกว้าง: อย่างน้อย 1920px
- ความสูง: อย่างน้อย 1080px
- อัตราส่วน: 16:9 หรือ 21:9
- ขนาดไฟล์: ไม่เกิน 2MB เพื่อการโหลดที่รวดเร็ว

### การแก้ไขรูปพื้นหลัง

หากต้องการเปลี่ยนรูปพื้นหลัง ให้แก้ไขใน `assets/css/style.css` บรรทัดที่ 32:

```css
background:
    linear-gradient(
        135deg,
        rgba(30, 64, 175, 0.7) 0%,
        rgba(59, 130, 246, 0.7) 50%,
        rgba(96, 165, 250, 0.7) 100%
    ),
    url("../images/your-image-name.jpg") center/cover no-repeat;
```

### การปรับแต่ง Overlay

คุณสามารถปรับความโปร่งใสของ overlay โดยเปลี่ยนค่า rgba:
- `rgba(30, 64, 175, 0.7)` - ค่า 0.7 คือความโปร่งใส (0.0-1.0)
- ลดค่าลงเพื่อให้รูปภาพเด่นมากขึ้น
- เพิ่มค่าขึ้นเพื่อให้ overlay เข้มขึ้น

### ตัวอย่างรูปภาพที่เหมาะสม
- รูปภาพที่มีแสงธรรมชาติดี
- รูปภาพที่มีพื้นที่ว่างสำหรับใส่ข้อความ
- รูปภาพที่เกี่ยวข้องกับธุรกิจ (อสังหาริมทรัพย์, เทคโนโลยี)
- หลีกเลี่ยงรูปภาพที่มีรายละเอียดมากเกินไป

### หมายเหตุ
- ระบบจะใช้ gradient overlay เพื่อให้ข้อความอ่านง่าย
- รูปภาพจะถูก crop ให้พอดีกับหน้าจอโดยอัตโนมัติ
- สำหรับ responsive design รูปภาพจะปรับขนาดตามหน้าจอ
